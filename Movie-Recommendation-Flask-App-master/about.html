🎬 About MovieMind

MovieMind is an AI-powered movie recommendation platform that suggests movies based on user preferences using collaborative filtering and machine learning techniques.

How MovieMind Works

Step 1: Enter the name of a movie and provide your personal rating (between 1 and 5).


Step 2: MovieMind searches a similarity matrix created from thousands of historical user ratings.


Step 3: Pearson Correlation is used to identify movies with similar user-rating behavior.


Step 4: The searched movie is excluded, and the top 8 most similar movies are recommended.

Technology Stack

Python Flask Pandas NumPy Scikit-Learn HTML5 CSS3

Recommendation Algorithm

MovieMind uses Item-Based Collaborative Filtering. It analyzes how users have rated different movies and computes similarity scores using Pearson Correlation. Movies with the highest similarity scores are ranked and the top 8 recommendations are displayed.
